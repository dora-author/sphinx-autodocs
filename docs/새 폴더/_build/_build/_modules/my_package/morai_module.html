[docs]    def arclength_of_line(self, x, y):
        """y=f(x)에서 매개 변수 x와 y로 둘러싸인 곡선의 길이를 구하는 함수이다.
        
        :param int x: x축 길이
        :param int y: y축 길이
        :returns arc: y=f(x)에서 x축와 y축으로 둘러싸인 전체 곡선의 길이
        
        **설명**: 각 x축과 y축 길이를 'num_segments'만큼 분할하고 피타고라스의 빗변 길이 공식을 활용하면 x축, y축의 분할 구간으로 둘러싸인 직선 거리를 구할 수 있다.
          이 직선 거리를 각 x축, y축의 분할 구간 개수인 'num_segments'만큼 구해서 모두 더한다.
          이때, 'num_segments'가 클수록 분할 구간의 간격이 작아지므로, 곡선에 가까운 직선의 거리를 구할 수 있다.

        **예제**:
            특정 인덱스에 대한 x축 분할 구간이 'x[indx] - x[indx - 1]'일 때 x와 y 분할 구간으로 둘러싸인 전체 곡선의 길이는 아래와 같이 구할 수 있다.

            >>> for indx in range(1, num_segments):
                    arc = arc + np.sqrt((x[indx] - x[indx - 1])**2 + (y[indx] - y[indx - 1])**2)
        """
        if len(x) != len(y):
            Logger.log_error('Check dimensions of the line')
            return
 
        num_segments = len(x)
        arc = 0
        
        for indx in range(1, num_segments):
            arc = arc + np.sqrt((x[indx] - x[indx - 1])**2 + (y[indx] - y[indx - 1])**2)
        
        return arc




[docs]    def coordinate_transform_point(self, angle, point):
        """회전축 X, Y를 갖는 2차원 좌표계에서 좌표 P가 세 번 회전했을 새로운 좌표 P'를 구하는 함수이다.

        :param int angle: 원점을 중심으로 한 회전각. 세 번 회전했을 때 각 회전각은 동일하다.
        :param tuple point: 기준 좌표. 좌표계에서는 벡터 p=(px,py)로 표시.
        :returns transform_pt: 새로운 좌표 P'

        **설명**: 세 번 회전했을 때 새로운 좌표는 회전 행렬의 내적을 이용하여 구한다.
            한 번 회전했을 때 회전 행렬은 아래와 같이 구할 수 있다:

            >>> rotation = np.array(((np.cos(angle), -np.sin(angle)),
            (np.sin(angle), np.cos(angle))))
            
        """
        rotation = np.array(((np.cos(angle), -np.sin(angle)),
            (np.sin(angle), np.cos(angle))))
 
        transform_pt = rotation.dot(point[0:2])
        
        return transform_pt
